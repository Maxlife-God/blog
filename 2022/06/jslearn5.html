赋值运算符

语法：变量 = 值;

赋值运算符 = 可以将符号右边的值赋给符号左边的变量。

a += 5 相当于 a = a + 5
a -= 5 相当于 a = a - 5
a *= 5 相当于 a = a * 5
a /= 5 相当于 a = a / 5
a %= 5 相当于 a = a % 5

关系运算符

通过关系运算符比较两个值之间的大小关系。

如果表达式成立则返回 true，如果不成立则返回 false

关系运算符：

> 大于号
>= 大于等于号
< 小于号
<= 小于等于号

规则：

  • 对于非数值进行比较时，会将其先转换成数值再进行比较
  • 任何值与 NaN 进行比较时都会返回 false
  • 如果两侧都是字符串，则不会将其转换为数值进行比较，而是会分别比较字符串中字符的 Unicode 编码：
    • 比较是一位一位对应比较的，如 "adb" > "bcd" 会先比较 a 和 b 的 Unicode 编码，若能得出结果，则直接返回结果，不再比较后面，这里 a 的编码小于 b 的编码，则表达式不成立，返回 false
    • 若两位相同，则比较下一位，直到能判断出结果
    • 在比较两个字符串类型的数值时，一定要转换数据类型

相等运算符

相等运算符 == 用来比较两个值是否相等。若相等则会返回 true，否则返回 false。

如果两个值的类型不同，则会自动转换为相同的类型之后再比较

不相等运算符 != 用来判断两个值是否不相等。若不相等则会返回 true，否则返回 false。

不相等运算也会自动进行类型转换。

全等 === 用来判断两个值是否全等，不同于「等于」的是，它不会自动进行类型转换，若两个值类型不同，则直接返回 false。

不全等 !== 用来判断两个值是否不全等，其同样不会进行类型转换，若两个值类型不同，则直接返回 true。

需要注意的是：

  • NaN 不和任何值相等，包括其本身。因此若需要判断一个值是否为 NaN 类型，需要使用 isNaN() 函数来判断，若参数为 NaN 类型则返回 true，否则返回 false
  • unidentified 衍生自 null，因此 unidentified == null 返回的结果为 true

条件运算符

条件运算符也叫三元运算符

语法：条件表达式?语句1:语句2;

首先会对条件表达式的结果进行判断，若结果为 true 则会执行语句1，否则会执行语句2。若条件表达式结果不为布尔值，则会先将其转换成布尔值再进行判断。
