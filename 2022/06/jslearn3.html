JS学习打卡-3日

发表于2022-06-20|更新于2022-11-12|技术
|字数总计:1.2k|阅读时长:4分钟|阅读量:

布尔值 Boolean

布尔值只有两个值：

  • true 真
  • false 假

Null 和 Undefined

Null（空值）类型的值只有一个 null，用来表示一个为空的对象。使用 tyepof 检查值为 null 的变量则会返回值 object。

Udefined（未定义）类型的值也只有一个 undefined

当变量声明了但未赋值时，输出变量则会输出值 undefined，使用 typeof 检查值为 undefined 的变量时也会返回 undefined。

强制类型转换

定义：将一种数据类型强制转换成另一个数据类型。一般是将其他数据类型转换成 String，Number，Boolean 类型。

将其他数据类型转换成 String 类型

方法一：调用被转换数据的 toString() 方法

调用方法的语法：变量名.方法名();

特点：该方法不会影响到原数据，只会将转换后的结果返回

缺陷：null 类型和 undefined 类型的值没有 toString() 方法，因此这两个类型的值不能调用该方法，否则会报错

方法二：调用 String() 函数

调用函数语法：函数名(参数);

特点：

  • 对于 Number 和 Boolean 类型的值实际上是调用 toString() 方法进行转换
  • 对于 Null 和 Undefined 类型的值则是直接转换（不调用 toString() 方法），会将 null 直接转换成 “null”，将 undefined 直接转换成 “undefined”

将其他数据类型转换成 Number 类型

方法一：调用 Number() 函数

特点：

  • 字符串转 Number 类型
    • 当字符串只含数字时，会将字符串直接转换为数字
    • 当字符串中含有非数字时，会将字符串转换成 NaN
    • 当字符串为空串或全为空格时，会将字符串转换成 0
  • Boolean 转 Number 类型
    • true 转换成 1
    • false 转换成 0
  • Null 转 Number 类型，会将其转换为 0
  • Undefined 转 Number 类型，会将其转换为 NaN

方法二：调用 parseInt() 和 parseFloat() 函数

注意：

  • 该方法专用于字符串
  • parseInt() 函数将字符串中的有效整数提取出来，然后转换成 Number 类型
  • parseFloat() 函数将字符串中的有效小数提取出来，然后转换成 Number 类型
  • 若传递的参数为非 string 类型，该函数会先将其转换成 string 类型再提取数字，因此 Boolean 类型和 Null 类型以及 Undefined 类型转换后的值都为 NaN
  • 对浮点数调用 parseInt() 函数会忽略掉小数部分，因此可以利用这一性质进行间接取整操作
  • parseInt() 函数可以传递第二个参数来指定转换的数字的进制，如 parseInt(a,10) 表示将变量 a 的值转换成 10 进制的数字

其他数据类型转换成 Boolean 类型

方法：使用 Boolean() 函数

特点：

  • 数字转 Boolean：除了 0 和 NaN，其余都转成 true
  • 字符串转 Boolean：除了空串，其余都转成 true
  • null 和 undefined 都转成 false
  • 对象也会转成 true

进制的表示方法

  • 二进制：以 0b 开头
  • 八进制：以 0 开头
  • 十六进制：以 0x 开头

二进制对浏览器的兼容性不好（虽然现在 ie 也无了~）
