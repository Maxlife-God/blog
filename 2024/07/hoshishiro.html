简单谈谈『星空列车与白的旅行』

发表于 2024-07-25 分类于 游戏

由于自己的文笔不是太好，也没怎么写过游戏评测，犹豫了许久，不知从何下笔，甚至不知道要说些什么，但最终还是写下了这些内容，权当记录自己的一些感受。以下内容涉及剧透，如想玩但还未玩过或未通关本作不建议观看。

本作的通关时长并不算长，相比于我玩的上一作『千恋万花』少了不少（毕竟是多线，本作只有一个结局），我算是推得慢的那种，前面没注意到的内容在后文再次提及时，我会返回前文再看一遍，因此比较耗时。不过刚体验完『千恋万花』的人性化的 UI 后，就觉得本作的 UI 真的是一大槽点，正如前面所说，我需要时不时返回某一个文本处再重新推一遍，但本作没有回溯文本的功能，仅仅能够看历史文本，而且能够看的历史文本似乎还有限制，太靠前的文本没法看。另外存档也是，只能够存但不能删，即只能覆盖存档，但游戏中不能把已有存档删除（直接删除存档文件应该是可行的）。

抛开上面的几个缺点来说，游戏的体验还是不错的。尤其是白玉老师的画风，当时就是冲着这个才来玩本作的我承认我是 llk，记得当时是被某粉红软件的星白的宣传视频的这一幕所吸引——

开始还以为「白」是指宣传图中的亚麻色发色的那个女孩（狩叶），直到晓和诺瓦第一次在星空下对话时，我才知道是男主的昵称。

尽管本作在 galgame 吧中依然有不少人称其剧情为「白开水」，我还是觉得很有意思。天生玩废萌的命。

剧情中多次提到一个概念——「世界是一个巨大的宝石箱，装满了满熠熠生辉的宝石」，最开始我以为男主说的他已经深深理解了这句话背后的意义是指夜空中的星星，毕竟看的宣传动画中列车在星空之下飞驰，后来才知道并非如此。世界是一个巨大的宝石箱，装满了满熠熠生辉的宝石，这些宝石就是「资源」，也可以说成是「资格」。小时候只要看着这些宝石就心满意足了，长大后就变得贪婪，变得想要获得这些宝石，但宝石是有限的，于是产生了「夺取」，我们想要获得宝石，就不得不与他人争夺。打个拙劣的比方，小时候我们都憧憬着「清华」「北大」等名校，觉得光是听到这个名字，知道这个学校就已经够了，但随着我们慢慢升学，我们会开始想去进入这些名校，但进入名校的资格是有限的，意味着我们必须与他人争夺。我是一个竞争性人格，即只要竞争我就能从中获取动力，只要在竞争中胜利我就会感到无比的快乐，没理解到这句话时我只是把它当做本作中一句煽情的话，仿佛要刻意营造一种意义深长的哲学的氛围，但其实它表达的意思很简单。我，不，应该是我们每个人，只要活在这个世界上，不管自己愿不愿意，都要去拿取这些宝石，因为这就是「成长」。

这里的「成长」不仅仅指身体上的成长，也指从认识、性格等感性的方面到工作等现实的方面。热爱动画的晓希望以后也能通过动画来维持生活，也曾憧憬过画出名作，也曾拼命地去夺取「宝石」，然后现实就如同给他泼了一桶冷水，曾一度想要放弃画画的他在音理的鼓励下才重新拿回自信。其他大人同样如此，怀揣着梦想来到大城市，才发现周围都是和自己一样的存在，自己不过是茫茫人海中的一个普通人，自己要与这些人去争夺「宝石」，会弄得自己遍体鳞伤，身心俱惫，有的人成功了，获取了宝石，但人是贪婪地，不会满足于手上已有的宝石，他们还会夺取更多的宝石；但也有的人失败了，他们没有获得宝石，甚至有的人连获取宝石的资格都没有了。这样一个过程就是一个人成长的过程。晓说自己已经不再感到宝石的价值了，即所谓的「被社会磨平了菱角」，怀揣着梦想从事着自己喜欢的工作，却渐渐对其失去了兴致，已经无法像以前那样充满热情地对待工作了，这真的很现实，现实中很少有人能把兴趣作为工作，如果能在这样的工作中一直保持着最初的热情就是更难的事了。大人们已成定局，但孩子们还有机会，他们还小，还有许多的时间去憧憬宝石，去努力争夺宝石，去体验各种美丽的事物，去感受「成长」。

感觉有点搞不清自己想说什么了，那就转来谈谈设定吧。本作的设定挺巧妙的，每个人都是诺瓦身体的一部分，已知的是狩叶是现实中猫村春香捐给夜羽真白的血液的化身，音理是移植到夜羽真白身上的心脏的化身，将移植到夜羽真白身上的器官人格化为其提供者，火车的锅炉代表了心脏，而这趟列车的终点就是心脏停止跳动。这趟列车本应没有终点，不，应该说是不应在现在迎来终点，即夜羽真白的生命还很漫长，她不应该在这个时候死去，正如晓所说，她应该去看更多更美的东西，去体验各种事物，她的旅途还很长。

作品中夜羽真白的母亲一直教导她的话——「不要给别人添麻烦」给我的印象也挺深刻的。由于是日专生，这个规则对我来说还是挺常见的，一般日本人在日常生活中都要遵循的便是这条规则，像是常见的「失礼ですが～」「お手数をおかけいたしまして申し訳ございません」，以及时时挂在嘴边的「すみません」，都是其语言方面的表现，再比如在电车中不要打电话，不要在骑车时听音乐等生活方面，都在践行着这一准则。因此这句话其实是符合现实中的日本的社会的规则的，而且结果反应出来是正确的，因为要注意不给别人添麻烦，所以要与他人保持一定的距离，因为有了距离感，人们才会顾及对方的感受，才会换位思考，这种距离感在一定程度上能够保持社会的和谐。但我同样认同晓所说的，即「既然不能给别人添麻烦，那就增加可以添麻烦的人」。乍一看前后是矛盾的，实则不然。正如前面所说，不给别人添麻烦实际上是与其他人保持一定的距离感，对外人来说这个距离非常远，因此是尽量不要给他人添加麻烦，而对于关系好的人，比如朋友、恋人、亲人等，保持过远的距离反而会显得生疏、见外，倒不如说他们更希望你能给他们添一些麻烦，这与外人不希望麻烦找上自己而你强行将麻烦推上去不同，这是当事人自己希望能够承受这份麻烦，希望能够帮助你，能够站在你的身旁，与你一同解决麻烦，因此在注意不要给他人添麻烦的同时，也要结交愿意与自己做朋友，愿意与自己拉近关系，自己能够依靠的人。

感觉自己说了一堆大道理
